STAR ACADEMY CYPRUS - Logo ve Giriş Görsel Paketi

İçerik:
- assets/staracademycy-logo.svg        : Açık zemin için logo
- assets/staracademycy-logo-white.svg  : Koyu zemin için logo
- assets/favicon.svg                   : Tarayıcı sekmesi / PWA ikonu
- assets/login-screen-reference.png    : İstediğin premium giriş ekranı referans görseli
- integration-snippet.html             : HTML ekleme kodları
- integration.css                      : CSS ekleme kodları

Kullanım:
1. assets klasöründeki dosyaları projenin assets klasörüne at.
2. index.html dosyasında <head> içine favicon satırını ekle.
3. Eski STAR ACADEMY yazılı logo alanını <img class="sac-logo" ...> ile değiştir.
4. CSS dosyana integration.css içeriğini ekle.
